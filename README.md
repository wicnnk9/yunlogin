## 浏览器指纹这东西，普通人真的不太了解

很多人以为换个IP就安全了，其实这只是冰山一角。平台检测账号关联的维度非常多：浏览器版本、操作系统、屏幕分辨率、语言设置、时区、字体列表、Canvas渲染特征、WebGL 参数、音频上下文……这些加在一起，组成了你设备独一无二的"数字指纹"。
